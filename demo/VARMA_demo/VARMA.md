> 姊妹篇：[../VAR_demo/VAR.md](../VAR_demo/VAR.md)、[../VMA_demo/VMA.md](../VMA_demo/VMA.md)——VARMA 是这两者的合体

## 0. 一句话理解 VARMA

**VARMA = VAR + VMA 合体**：右边**既有看得见的过去的值**（AR 部分），**又有看不见的过去的冲击**（MA 部分）。

$$
r_t=\phi_0+\underbrace{\Phi_1 r_{t-1}+\dots+\Phi_p r_{t-p}}_{\text{AR：长记忆，值层层递推}}+a_t\underbrace{-\Theta_1 a_{t-1}-\dots-\Theta_q a_{t-q}}_{\text{MA：有限记忆，冲击只活 }q\text{ 期}}
$$

它同时背着两种记忆：**AR 拖很长**（几何衰减回均值），**MA 只活 $q$ 期**（之后失效）。$\Sigma$ 的非对角仍是同期相关。

> **但请先记住书的态度**：金融里 VAR 或 VMA 单用基本就够，VARMA 强大却**难驾驭**（识别问题），所以**能不用就不用**，要用也只取低阶 $(1,1)/(2,1)$。本 demo 就是带你走一遍"为什么慎用、真要用怎么走"。

## 1. 与 VAR / VMA 的对照（精华，整篇就记这张表）

| 步骤 | VAR | VMA | **VARMA（合体）** |
|---|---|---|---|
| ⓪ 前提 | 直接上 | 直接上 | **多一道"劝退关"**：VAR/VMA 不够才上 |
| ① 定阶 | CCM 拖尾→AIC/BIC | CCM 截尾→数 $q$ | CCM 也拖尾→**还要解决"模型不唯一"（识别）** |
| ② 估计 | OLS | MLE（反推冲击） | **MLE**（含 MA→反推冲击）+ 识别约束 |
| ③ 检验 | $Q_k(m)$+CCM | 同 | **完全一样** |
| ④ 预测 | 长记忆，慢衰减 | 有限记忆，$>q$ 即回均值 | **两种记忆叠加**：短期 MA 修正 + 长期 AR 衰减回均值 |

> 一句话：**VARMA 的四步骨架和 VAR/VMA 一样，但第①步多了一道"识别（解决唯一性）"的坎——这是它和 VAR/VMA 最大的不同，也是书不展开、实务慎用的根本原因。**

---

## 2. 核心概念速查

| 概念 | 含义 |
|---|---|
| 公因子 common factor | AR 与 MA 多项式含同一个因子 $(1-cB)$，可**相消** → 同一过程有无数等价写法 |
| 不可识别 non-identification | 因公因子等，参数 $(\Phi,\Theta)$ 不唯一；CCM/数据分辨不出 |
| 结构指定 structural spec. | Tiao-Tsay(1989)/Tsay(1991)：系统性去公因子、把模型钉唯一 |
| Kronecker 指数 / SCM | 结构指定的两套技术（标量分量模型）；找出"哪些方向放 AR、哪些放 MA" |
| 正规化 normalization | 本 demo 用的实用约束：$\Phi_0=\Theta_0=I$ + 取**最低阶** |
| $\Psi$ 权重 | MA($\infty$) 表示 $=\Phi(B)^{-1}\Theta(B)$ 的系数，**脉冲响应**；它是**可识别的不变量** |
| 弱识别 weak ident. | $\Phi$ 与 $\Theta$ 特征值**接近**→近公因子→似然平坦→MLE 乱跑 |
| Hannan-Rissanen | 两步法给 MLE 一个**一致的暖启动**，避免落入等价/坏解 |

**关键认知**：VARMA 真正被数据钉死的是 **$\Psi$ 权重（脉冲响应）**，不是 $(\Phi,\Theta)$ 这对矩阵本身。同一组 $\Psi$ 可由多对 $(\Phi,\Theta)$ 生成（公因子）→ 必须靠**结构约束**才能从 $\Psi$ 反推出唯一的 $(\Phi,\Theta)$。

---

## 3. 数据要求（喂进去之前先验收）

| 要求 | 怎么查 | 不满足怎么办 |
|---|---|---|
| **确实非用 VARMA 不可** | VAR/VMA 先试 | VMA 能截尾→用 VMA；低阶 VAR 能白→用 VAR |
| **平稳** | AR 部分 $\Phi$ 特征值 < 1 | 差分 / 协整 |
| **可逆** | MA 部分 $\Theta$ 特征值 < 1 | 冲击反推不回 → 加可逆约束 |
| **可识别（无公因子）** | $\Phi,\Theta$ 特征值别太近；结构指定 | 近公因子→弱识别→换约束 / 降阶 |
| **低阶** | 取 $(1,1)/(2,1)$ | 高阶 VARMA 几乎没法稳健估 |

> 一句话：**喂进 VARMA 的应是——VAR/VMA 都搞不定、平稳且可逆、AR 与 MA 没有近公因子（强可识别）、且能用低阶 $(1,1)/(2,1)$ 拿下的多元序列。**

## 4. 五步流程（对照 demo）

```
VARMA(1,1)造数据 ─→ ⓪劝退关 + 前提 ─→ ①定阶+识别 ─→ ②估计 ─→ ③检验 ─→ ④预测
                   VMA数不出q&VAR(1)不白  公因子致不唯一    Hannan-Rissanen   Q_k+CCM   短期MA修正
                   →才上VARMA(低阶)      →正规化钉(1,1)    暖启动+条件MLE    (同前)    +长期AR衰减回均值
```

### ⓪ 第1步 劝退关 + 前提（VAR/VMA 没有的关卡）
书的建议先落地：**先看 VAR/VMA 够不够**。本 demo 两条路都堵死才上 VARMA：
- **VMA 路线**：样本 CCM **缓慢拖尾、不归零**（显著格数 `[3,2,2,1,0,...]`）→ 数不出截尾阶 $q$ → 纯 VMA 不适用。
- **VAR 路线**：纯 **VAR(1) 残差 $Q_k$ 仍显著**（$p\approx0.03$）→ 要堆很高阶才白（MA = 无穷阶 AR）→ 不省参数。
- 两者特征都有 → 才上 **VARMA(1,1)**（只取低阶）。
- 前提同前：AR 特征值 < 1（平稳）、MA 特征值 < 1（可逆）。

### ① 第2步 定阶 + 识别（VARMA 独有、最难的坎）
VARMA 的 CCM 像 VAR 一样拖尾 → 光看 CCM **定不出 $(p,q)$**；更糟的是**模型不唯一**：

- **识别问题（公因子）**：给 AR、MA **同乘一个公因子** $(1-cB)$，VARMA(1,1) 就变成一个 VARMA(2,2)，二者产生**完全相同**的 $r_t$。本 demo 用 $c=0.3$ 构造这个 (2,2)，算出两者的 $\Psi$ 权重（脉冲响应）**逐阶最大差 $\approx 5\times10^{-17}$**——数值上一模一样 → 同一过程的两种写法 → 数据分不开。
- **解决（结构指定）**：Tiao-Tsay(1989)/Tsay(1991) 的 **Kronecker 指数 / 标量分量模型（SCM）** 系统性去公因子。这是最专业、最易翻车的一步，书因此点到为止。
- **本 demo 的实用正规化**：$\Phi_0=\Theta_0=I$（已隐含）+ 取**最低阶** → 把 $(p,q)$ 钉死在 $(1,1)$。

### ② 第3步 估计（条件 MLE，且要暖启动）
含 MA → 冲击看不见 → 不能纯 OLS，只能 **MLE**（同 VMA）。VARMA 还有个坑：**弱识别会让 MLE 乱跑**（$\Phi\approx\Theta$ 时近公因子、似然平坦）。本 demo 两手准备：

1. **Hannan-Rissanen 两步暖启动**（一致估计，防落入等价坏解）：① 先拟合一个**长阶 VAR** 取残差当冲击 $a$ 的代理；② 再把 $r_t$ 对 $[1,r_{t-1},\hat a_{t-1}]$ 做 OLS，得到 $(\theta_0,\Phi_1,\Theta_1)$ 初值（$\hat a_{t-1}$ 的系数 $=-\Theta_1$）。
2. **条件 MLE**：从暖启动出发，$\Sigma$ 解析集中并出，优化 $\theta_0,\Phi_1,\Theta_1$ 共 10 个参数。
3. **精修**：剔 $|t|<1.96$ 的项再估 → 本 demo 把 MKT 行的两个 STK 项（AR、MA 各一个）剔为 0，**还原"双重单向"**。

> demo 里特意把真值设成 $\Phi_1$ 特征值 $\{0.6,0.5\}$（正）、$\Theta_1$ 特征值 $\{-0.4,-0.3\}$（负）——**远离、不相消**，强可识别。若故意让 $\Phi\approx\Theta$，MLE 会跑到一个残差同样白、但参数离谱的等价解（这正是弱识别的真实写照）。

### ③ 第4步 检验（与 VAR/VMA 完全相同）
残差跑**多元 Ljung-Box $Q_k(m)$**（$df=k^2m-k^2(p+q)$）+ **CCM 符号表**（几乎全 `.`）。不合格 → 回第①步调 $(p,q)$ 或重新识别。（个别格偶然显著属抽样噪声，以 $Q_k$ 为准。）

### ④ 第5步 预测（两种记忆叠加）
- $\hat r_T(1)=\theta_0+\Phi_1 r_T-\Theta_1 a_T$（**AR 项 + 尚存的 MA 冲击修正**）；
- $\hat r_T(\ell)=\theta_0+\Phi_1\hat r_T(\ell-1)$，当 $\ell>q$（**MA 失效，纯 AR 递推**）→ 几何衰减回均值 $\mu=(I-\Phi_1)^{-1}\theta_0$；
- 误差协方差 $\hat\Sigma(\ell)=\sum_{j=0}^{\ell-1}\Psi_j\Sigma\Psi_j'$（$\Psi$ 由估计模型算）。
- **综合效果**：短期受冲击项影响、长期由 AR 主导慢慢衰减回均值——这正是 VAR（无 MA 修正）与 VMA（$>q$ 直接拍平）之间的"混血"。见 `varma_fig5` 右图：$\|\Psi_\ell\|$ 在 $\ell\le q$ 含 MA 修正、之后 AR 几何衰减。

## 5. 判据一览

| 步骤 | 判据 | 通过条件 |
|---|---|---|
| 要不要 VARMA | VMA 能否截尾 / 低阶 VAR 是否白 | 都不行 → 才上 |
| 平稳 / 可逆 | $\Phi$ / $\Theta$ 特征值模长 | 全 < 1 |
| 识别 | $\Psi$ 权重等价性 + 结构约束 | 加约束后 $(\Phi,\Theta)$ 唯一 |
| 估计 | Hannan-Rissanen 暖启动 + 条件 MLE 收敛 | 似然最大、参数稳定 |
| 残差白噪声 | 多元 Ljung-Box $Q_k(m)$ | > 0.05 |
| 残差无残留 | CCM 符号表 | 几乎全 `.` |
| 预测·大小 | RMSE vs 朴素 | 更小 / 相当 |
| 预测·分布 | 区间覆盖率 | ≈ 95% |

## 6. 易混点澄清

1. **VARMA 的不唯一 vs VAR/VMA 的唯一**：VAR、VMA 在 $\Phi_0=\Theta_0=I$ 下天然可识别；VARMA 因 AR、MA 可含**公因子**而**不唯一**——这是它独有的坎，必须额外加结构约束。
2. **被数据钉死的是 $\Psi$，不是 $(\Phi,\Theta)$**：多对 $(\Phi,\Theta)$ 给同一 $\Psi$（同一过程）。所以"识别"= 从 $\Psi$ 里挑出唯一的 $(\Phi,\Theta)$。
3. **弱识别 ≠ 不可识别**：$\Phi\approx\Theta$（近公因子）时模型**形式上可识别但似然极平**，MLE 数值上会乱跑 → 需 Hannan-Rissanen 暖启动 + 避免近公因子的设定。
4. **为什么要暖启动**：直接从 $\Theta=0$ 起步，MLE 可能走进残差同样白、但参数离谱的等价解；Hannan-Rissanen 给一个**靠近真值的一致初值**，把 MLE 引到对的盆地。
5. **两种记忆**：MA 给短期一次性修正（只活 $q$ 期）、AR 给长期几何衰减。别指望 VARMA 的 MA 部分管长程——长程永远是 AR 说了算。
6. **能不用就不用**：书反复强调金融里 VAR/VMA 基本够用，VARMA 收益有限、风险（识别翻车）很大，只在确有必要时上低阶。

## 8. 全流程记忆口诀

> **先过"劝退关"（VAR/VMA 都不够才上 VARMA，且只取低阶）→ 定阶撞上"识别坎"（CCM 拖尾定不出、AR/MA 公因子致不唯一，用结构指定/正规化钉死 $(p,q)$）→ Hannan-Rissanen 暖启动 + 条件 MLE 反推冲击估参、精修剔单向 → 残差查 $Q_k(m)$ 与 CCM → 两种记忆预测（短期 MA 修正、长期 AR 衰减回均值）。**
> 骨架与 VAR/VMA 同；**唯一的大不同是第①步要"解决唯一性"**——这也是 VARMA 强大却难用、书里点到为止的根源。
