# TAR / SETAR 门限自回归建模 —— 学习总结

> 配套代码：[tar_demo.py](tar_demo.py)（一条龙六步，门限按"正负号"分两状态）
> 数据/产物：`tar_data.csv`（输入序列+状态标签）、`tar_forecast.csv`（预测输出+方向列）、`tar_fig1~5_*.png`

---

## 0. 一句话理解 TAR

普通 AR 全程只有**一套**系数，刻画的是"线性、对称"的动态。但很多真实序列是**分状态**的——涨势里和跌势里、扩张期和收缩期，运动规律根本不一样。

**TAR（门限自回归，Threshold AR，Tong 1978）就是把 AR 按"状态"切成几段，每段用各自的一套 AR 系数。** 用哪一段，由一个**门限变量**跨没跨过**门限值**来决定。最常用的 **SETAR**（自激励 TAR）直接用序列**自己的滞后值** `y_{t-d}` 当门限变量。

- **AR 管"一套线性规律"**；
- **TAR 管"分状态、各一套规律"**——条件均值在门限处有**跳变/折点**，是分段线性的非线性模型。

本 demo 把门限定成最直观的**正负号**：`d=1, r=0`——**上一步为负走一套、为正走另一套**。

## 1. 核心概念速查

| 概念 | 含义 |
|---|---|
| 门限变量 threshold variable | 决定当前处于哪个状态的量；SETAR 用序列自身滞后 `y_{t-d}` |
| 延迟 delay $d$ | 用第几阶滞后当门限变量（本 demo `d=1`，看上一步） |
| 门限值 threshold $r$ | 切分状态的分界线（本 demo `r=0`，即正负号） |
| 状态 / 区制 regime | 被门限切出来的若干段，每段一套 AR 系数（本 demo 2 段：L 负、H 正） |
| 局部中心 | 单段的吸引点 $c^{(j)}/(1-\phi^{(j)})$，序列在该状态下围绕它波动 |
| 自激励 self-exciting | 门限变量取自序列自身（而非外生变量）→ SETAR |
| 极限环 / 非对称周期 | 两状态切换形成的"急涨缓跌"等非对称循环，线性 AR 做不出 |

**关键认知**：TAR 的非线性**藏在条件均值里**——`E[y_t | y_{t-1}]` 是一条在 `r` 处**断开/拐折**的折线，而不是一条直线。一条直线（线性 AR）跨不过这个台阶，这就是要用 TAR 的根本理由。

### 模型方程（2 状态 SETAR(2; 1,1)，本 demo 真值）

$$
y_t=\begin{cases}
c^{(L)}+\phi^{(L)}y_{t-1}+\varepsilon_t, & y_{t-d}\le r\quad(\text{状态 L})\\[4pt]
c^{(H)}+\phi^{(H)}y_{t-1}+\varepsilon_t, & y_{t-d}> r\quad(\text{状态 H})
\end{cases}
$$

本 demo 真值（`d=1, r=0`）：

$$
y_{t-1}\le 0:\ y_t=-3.0+0.5\,y_{t-1}+\varepsilon_t\quad(\text{中心}-6.0,\ \text{AR 更强、更黏})
$$
$$
y_{t-1}> 0:\ y_t=+4.0+0.3\,y_{t-1}+\varepsilon_t\quad(\text{中心}+5.7,\ \text{AR 更弱、更短})
$$

两段**中心一负一正、分得开** → 序列在正/负两个"电平"间切换；段内**弱 AR**（0.5 / 0.3）→ 同号会**黏成一小段**。这正是"两个离散状态、各有弱自相关"的形态。

## 2. 数据要求（喂进去之前先验收）

| 要求 | 怎么查 | 不满足怎么办 |
|---|---|---|
| **全局平稳** | ADF / KPSS | 先差分或变换；注意：单段可不平稳，整体仍可平稳 |
| **门限变量、延迟 $d$ 明确** | 业务知识 / 网格搜 $d$ | 未知就在 $d=1..D$ 上搜，按 SSR 最小定 |
| **门限值 $r$ 明确或可搜** | 业务（本例正负号 $r=0$）/ 网格搜 $r$ | 未知 → 条件最小二乘网格搜（见步骤3） |
| **每个状态样本足够** | 数各段观测数 | 门限太极端→某段太少→**修剪**到 15%~85% 分位再搜 |
| **确有门限效应** | 门限 F / Tsay / Hansen 检验 | 不显著 → 退回线性 AR，别强上 TAR |
| **等间隔、有序、无大缺口** | 看时间索引 | 重排、补齐或分段 |
| **噪声分布兜得住** | 残差直方图 / QQ | 厚尾换 t / 偏 t |

> 一句话：**喂进 TAR 的应是——等间隔、整体平稳、门限变量与状态划分清楚（或可由数据搜出）、且每个状态都有足够样本、确实存在门限效应的序列。**

## 3. 六步建模流程（对照 demo）

```
SETAR 造数据 ──→ ADF平稳 + 线性AR(基准) ──→ 门限非线性检验(要不要TAR)
                       │                          │ Tsay TAR-F(门限未知,主) + Chow-F(门限已知,简化确认)
                       ↓                          ↓ 显著→上TAR
   蒙特卡洛预测 ←── 残差白噪声检验 ←── 分段OLS估计 ←── 识别 d,r(网格搜)+ p(各段AIC)
 (点+区间,远期双峰;三档评估)  (合并残差Ljung-Box)  (各段c,φ,σ)
```

### 第0步 准备数据
用 SETAR 真值造序列，留尾部做样本外。验全局平稳（ADF）。

### 第1步 平稳性 + 线性 AR 的"破绽"
ADF 验平稳。**先拟合一个线性 AR(1) 当基准**——它把正负两段强行压成一条直线，必然抓不住 0 处的台阶。这条基准就是下一步门限检验的"受限模型"。

### 第2步 门限非线性检验（要不要 TAR）

门限检验有两套，关键看**门限 `r` 是已知还是要从数据估**——这决定了 F 还服不服从标准分布。本 demo 两套都做：**(2a) 以 Tsay 检验为主**（书本 §4.2 的正统做法，不假设你知道 `r`），**(2b) 再用 Chow-F 在"正负号已知"前提下确认一遍**。

**(2a) Tsay(1989) 排列自回归 TAR-F（门限未知时的标准检验，本 demo 主检验）**

门限未知时，零假设（线性）下 `r` 不可识别 → **Davies 问题** → 普通 F 不再服从标准 F 分布。Tsay 用一个巧妙的"排序+递归"把它绕开：

1. **排列自回归（arranged autoregression）**：把回归行 `{y_t; 1, y_{t-1},…,y_{t-p}}` 按**门限变量 `y_{t-d}` 升序**重排。若真有门限，重排后"前一段（小 `y_{t-d}`）"和"后一段（大 `y_{t-d}`）"会服从**不同**的 AR。
2. **递归最小二乘 → 标准化预测残差**：沿重排顺序逐点递归估计，对下一个点做一步预测，取**标准化预测残差** $\hat e_m=(y_m-x_m'\hat\beta_{m-1})/\sqrt{1+x_m'(X'X)^{-1}x_m}$。线性下这些残差应是白噪声、**与过去无关**。
3. **辅助回归取 F**：把 $\hat e_m$ 对回归元 $[1,y_{t-1},…,y_{t-p}]$ 回归，检验系数是否全 0：

$$F(p,d)=\frac{\big(\textstyle\sum\hat e_m^2-\sum\hat\varepsilon_m^2\big)/(p+1)}{\sum\hat\varepsilon_m^2/(n_{\text{eff}}-p-1)}\ \sim\ F(p+1,\ n_{\text{eff}}-p-1)$$

`F` 大 → 残差仍与过去相关 → 拒绝线性、**存在门限**。这个检验**额外的好处**：对 `d=1,2,…` 各算一个 `F`，**F 最大的 `d` 就是推荐的延迟**（本 demo 在 `d=1` 处 F 最大且 `p≈0.002`，正好挑出真延迟）。

**(2b) Chow / 结构突变 F（门限已知 `r=0` 时的简化确认）**

本 demo 的门限是"正负号"这个**外生设定**（`r=0` 事先定死），既然 `r` 不用估，就没有 Davies 问题，可直接比"线性 AR（受限）"与"两段 TAR（非受限）"：

$$F=\frac{(SSR_0-SSR_1)/(k_1-k_0)}{SSR_1/(n-k_1)}\ \sim\ F(k_1-k_0,\ n-k_1)$$

它和 (2a) 结论一致（极显著）。**两点注意**：

- Chow-F 的标准分布**假设两段同方差**（本 demo 真值两段 `σ` 都=4.5，不冲突）；若两段方差确不同，严格应改用**异方差稳健（White）版**的 Wald 统计量。
- 这个标准 F **只在门限已知时成立**；门限未知就回到 (2a) 的 **Tsay(1989) / Hansen(1996,1999) 自助法**。

p 小 → 门限效应显著，上 TAR；不显著 → 老实用线性 AR。

### 第3步 识别 `d, r, p`（书 §4.3 要**同时**定这三样）

**(3a) 门限 `r` & 延迟 `d`（网格搜）**：即便按正负号已知 `r=0, d=1`，也**网格搜一遍**验证数据自己也这么选：

- 对每个候选 `d=1..D`、每个候选门限 `r`（取在门限变量的 15%~85% 分位之间，保证两段样本足够），**按 `y_{t-d}≤r` 切两段各做 OLS**，记总 `SSR(d,r)`；
- 取使总 SSR 最小的 `(d,r)`（**条件最小二乘**）。本 demo 搜出的最优正好落在 `d=1, r≈0` —— 与"正负号"一致（见 `tar_fig2`：SSR 曲线在 0 处尖锐见底）。

**(3b) 段内 AR 阶 `p`（各段 AIC 选，可不同阶）**：定门限时先用基准阶（这里 1）搜 `(d,r)`；定下 `(d,r)` 后，**对每个状态各自**在 `p=1..P` 上算 AIC，**独立**取最小者——所以 L 段、H 段**可以有不同的阶 `pL, pH`**。本 demo 两段 AIC 都在 `p=1` 见底（真值正是 1 阶），故 `pL=pH=1`。
> 要点：阶 `p` 是**选**出来的、不是写死的。下游估计/残差/预测全部按选出的 `(pL,pH)` 走，换个数据若选出 `p>1` 也照样跑。

### 第4步 分段估计（条件 OLS）
给定 `d, r`，TAR 估计**极其简单：按状态把样本切开，每段各跑一次 OLS**。报每段 `c, φ` 与真值对照、每段残差 `σ`（TAR 允许两段方差不同）。

### 第5步 模型检验（建得够不够好）
把两段残差按时间合并，**Ljung-Box** 查是否白噪声：

| 查什么 | 工具 | 不过关 → |
|---|---|---|
| 残差无自相关 | 合并残差 Ljung-Box | 段内 AR 升阶 / 改门限 |
| 分布假设 | 偏度、峰度、QQ | 换 t / 偏 t 噪声 |

过关 → 模型充分；不过关 → 回上面改，迭代。

### 第6步 使用模型（预测）—— TAR 与 AR 最大的不同
- **1 步预测**：已知 `y_t` 就知道落哪段，直接代公式，确定。
- **多步预测**：因为 `E[g(y)]≠g(E[y])`，**不能简单递推**——未来落哪段是随机的。标准做法是**蒙特卡洛模拟**：从当前值出发，按估计的 TAR + 噪声模拟大量路径，
  - **点预测** = 各步路径的均值（会非线性地回复到全局均值）；
  - **预测区间** = 各步路径的分位数（可不对称）；
  - **预测密度**：短期≈单峰（贴近起始状态），**远期变双峰**（两个状态各一峰）→ 见 `tar_fig4` 下图，这是 TAR 区别于 AR 的最直观证据。

**样本外评估三档**（书 §4.4.2 的标准三档：方向 / 大小 / 分布，缺一不全）：

| 档 | 指标 | 本 demo 做法 / 结果 |
|---|---|---|
| **大小** | MSE / RMSE | 点预测 RMSE 显著优于 last-value 朴素基准 |
| **分布** | 区间覆盖率 | 95% 蒙特卡洛区间的实际覆盖率≈95% |
| **方向** | **2×2 命中表 + χ² 检验** | 用各步 `P(y>0)` 多数表决预测**区制（正负号）**，与实际比 2×2 列联表，χ² 检验是否**优于"抛硬币"**（恒猜多数类）。本 demo 命中率≈82% > 基准 60%，`χ²≈16.7, p≈4e-5`，方向显著可预测 |

> 方向档是书里得出"神经网络没赢过抛硬币"那个著名结论用的工具——**光看 RMSE 小还不够，得看方向猜没猜对**。本 demo 因为区制就是正负号、短期高度可预测，所以**赢过了抛硬币**；但看 `tar_fig5` 右图，`P(y>0)` 随步长从 ~0.95 衰减到 ~0.44（全局比例），**远期方向终归回到基准**——和书里"长程难"的精神一致。

> 见 [tar_forecast.csv](tar_forecast.csv)：每行 = 未来一步的 `mc_mean`(点) + `lo95/hi95`(区间) + `actual`(实际) + `p_up/pred_pos/actual_pos`(方向)。区间由模拟路径分位给出，能容纳"可能已切到另一状态"的不确定性，这是线性 AR 的对称区间做不到的。

## 4. 判据一览（每步用什么 p 值 / 准则决策）

| 步骤 | 判据 | 通过条件 |
|---|---|---|
| 全局平稳 | ADF p | < 0.05 |
| 有无门限效应 | **Tsay(1989) TAR-F**（门限未知，主）/ Chow-F（门限已知，简化）/ Hansen 自助 p | < 0.05 → 有 |
| 延迟 $d$ | Tsay TAR-F 按 $d$ 比大小 / 网格搜总 SSR | F 最大 / SSR 最小 |
| 门限 $r$ | 条件最小二乘：总 SSR | 取最小 |
| 段内 AR 阶 $p$（各段独立） | 各段 AIC / BIC | 准则最小 |
| 残差白噪声 | 合并残差 Ljung-Box p | > 0.05 → 充分 |
| 预测·大小 | RMSE / MSE vs 朴素基准 | 更小 → 更优 |
| 预测·分布 | 区间覆盖率 | ≈ 名义水平(95%) |
| 预测·方向 | 区制命中 2×2 + χ² | p < 0.05 → 优于抛硬币 |

## 5. 易混点澄清

1. **TAR vs AR**：AR 一条直线（线性、对称）；TAR 是在门限处**断开的折线**（分段线性、可不对称）。判断要不要 TAR，看条件均值有没有"台阶/拐折"——即门限效应检验。
2. **TAR vs STAR**：TAR 是**硬切换**（跨过门限瞬间换系数，"离散状态"）；STAR（平滑过渡）用一条 S 形权重在两套系数间**渐变**。本 demo 是硬切换。
3. **TAR vs Markov 切换(MS)**：TAR 的状态由**可观测的门限变量**决定（看 `y_{t-1}` 正负即知）；MS 的状态是**不可观测的隐变量**，要靠滤波推断。
4. **单段不平稳 ≠ 整体不平稳**：TAR 一个经典结论——**某段即使含单位根甚至局部发散，只要切换机制把序列推回，整体仍可全局平稳**（遍历性条件见 Petruccelli-Woolford 1984；本 demo 满足 `φ_L<1, φ_H<1, φ_L·φ_H<1`）。
5. **门限未知时不能用标准 F 分布**：零假设下门限 `r` 不可识别（Davies 问题），p 值要走 **Tsay(1989) 排列自回归 / Hansen 自助**——这正是本 demo 第 2 步 **(2a)** 跑的主检验（它不假设你知道 `r`，还顺带挑出延迟 `d`）。**(2b)** 的标准 Chow-F **只在门限已知（这里正负号 `r=0` 外生定死）时才成立**，是个简化确认；且它假设两段同方差，方差不同应改异方差稳健版。**别把"门限已知的 Chow-F"当成书里那个"门限未知的 TAR-F"**——两者算法不同，只是本例结论一致。
6. **多步预测不是代一次公式**：非线性下必须蒙特卡洛模拟取均值/分位，且远期预测密度会**双峰**。

## 7. 全流程记忆口诀

> **整体平稳的序列 → 先用线性 AR 当基准（看它跨不过门限台阶）→ 门限非线性检验定"要不要 TAR"（门限未知用 Tsay TAR-F 为主、已知再用 Chow-F 确认）→ 识别 `d,r`（网格搜）与段内阶 `p`（各段 AIC）→ 按状态切开各跑一次 OLS → 合并残差查白噪声、不行就迭代 → 蒙特卡洛模拟做多步预测（点+区间，远期双峰），并按"大小/分布/方向"三档评估（方向看区制命中是否赢过抛硬币）。**
> 每一步的"门槛"都是一个假设检验的 p 值或信息准则，不达标就回上一步。
