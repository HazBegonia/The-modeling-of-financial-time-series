## 0. 一句话理解 VMA

**VMA（向量滑动平均）= 把 VAR 右边"看得见的过去的值"换成"看不见的过去的冲击"。**

$$
\underbrace{r_t}_{k\times1}=\underbrace{\theta_0}_{\text{均值}}+a_t-\Theta_1 a_{t-1}-\dots-\Theta_q a_{t-q},\qquad \mathrm{Cov}(a_t)=\Sigma
$$

- $\Theta_i$ 的对角线 = 每个序列对自己过去冲击的反应；
- $\Theta_i$ 的非对角线 = "谁的冲击传给谁"的**跨序列冲击传导**；
- $\Sigma$ 的非对角线 = "同一天一起来"的**同期相关**（和 VAR 一样，仍藏在噪声协方差里）。

本 demo 用 $k=2$（个股 `STK` + 大盘 `MKT`），真值 $\Theta_1$ 的 $(2,1)$ 元设为 0 —— **STK 的过去冲击不传给 MKT**，复刻"单向传导"，便于演示精修把它剔回 0。

---

## 1. 与 VAR 的对照（精华，全篇就记这张表）

| 步骤 | VAR | VMA | 根源 |
|---|---|---|---|
| ① 定阶 | CCM **拖尾** → 靠 AIC/BIC | 数 CCM **截尾**定 $q$（$\rho_l=0,\ l>q$） | MA 的相关有限长 |
| ② 估计 | 右边是过去的**值** → **OLS** | 右边是过去的**冲击**（看不见）→ 只能 **MLE 反推** | 冲击不可观测 |
| ③ 检验 | $Q_k(m)$ + 残差 CCM | **完全一样** | — |
| ④ 预测 | 值层层递推 → 记忆**长** | 超过 $q$ 步即回均值 $\theta_0$ → 记忆**有限** | 只用最近 $q$ 期冲击 |

> 一句话：**VMA 建模骨架和 VAR 一样，只有"定阶靠 CCM 截尾"和"估计只能 MLE 反推冲击"两处不同，根源都在"冲击看不见"。**

---

## 2. 核心概念速查

| 概念 | 含义 |
|---|---|
| 冲击 $a_t$ | 看不见的新息（innovation）；VMA 的右边全靠它 |
| 截尾 cut off | 样本 CCM 在第 $q$ 阶后**骤降为 0**（VAR 是缓慢拖尾 tail off） |
| 可逆性 invertibility | $\Theta(B)$ 的根在单位圆外 ⇔ $\Theta_i$ 系数矩阵特征值在单位圆**内**；可逆才能由 $r$ 唯一反推冲击 $a$ |
| 条件 MLE | 先令初始冲击 $a_0=0$，递推 $\hat a_t=r_t-\theta_0+\Theta_1\hat a_{t-1}+\dots$，再让高斯似然最大 |
| 精确 MLE | 不把 $a_0$ 当 0，而当未知量一起估，更准更重 |
| 可识别性 | VMA 参数可能不唯一（如符号/尺度），需加约束 |
| 有限记忆 | 预测步数 $>q$ 时冲击全用完 → 预测值=均值 $\theta_0$、误差方差不再增长 |

**关键认知**：VMA 的相关结构是**有限长**的——只有最近 $q$ 期冲击在起作用。这一条同时解释了它的三个特征：**CCM 在 $q$ 后截尾**、**预测在 $q$ 后回均值**、**预测误差方差在 $q$ 后封顶**。

## 3. 数据要求（喂进去之前先验收）

| 要求 | 怎么查 | 不满足怎么办 |
|---|---|---|
| **平稳** | 各分量 ADF | 有限阶 VMA **恒平稳**，一般不必担心 |
| **可逆** | $\Theta_i$ 特征值模长 < 1 | 不可逆 → 冲击反推不回、MLE 无解 → 加可逆约束 |
| **CCM 确有截尾** | 样本 CCM 符号表 | 若拖尾不截 → 其实该用 VAR / VARMA |
| **可识别（加约束）** | 看参数是否唯一 | 加约束（如下三角、标准化）破除多解 |
| **维数 $k$ 不爆炸** | 数 $k$；参数 $\propto k^2 q$ | $k$ 大 → 稀疏化 / 因子模型 |

> 一句话：**喂进 VMA 的应是——平稳（自动满足）、可逆（冲击反推得回）、样本 CCM 看得出截尾、且参数可识别的多元序列。**

## 4. 五步建模流程（对照 demo）

```
VMA(1)造数据 ─→ 前提:各分量ADF + 可逆性(Theta特征值<1) ─→ 定阶 q
                                                          │ 数 CCM 截尾(招牌) —— 不靠 AIC/BIC
                                                          ↓
      有限记忆预测 ←──── 残差检验 ←──────────────── 估计
   (>q步回均值,误差封顶)  Q_k(m)+CCM(与VAR同)   条件MLE反推冲击 + 精修剔不显著 + 可识别约束
```

### 前提 第1步 平稳 / 可逆
有限阶 VMA **必平稳**，ADF 只是走个过场。真正要查的是**可逆性**：$\Theta_1$ 特征值模长全 $<1$，冲击 $a$ 才能由 $r$ **唯一反推**——这是下一步 MLE 能跑的前提。本 demo $\Theta_1$ 特征值 $\{0.5,0.4\}$，可逆。

### 定阶 第2步 数 CCM 截尾定 $q$（VMA 招牌）
VMA 的样本 CCM 在第 $q$ 阶后**截尾**（$\rho_l=0,\ l>q$）。做法：把样本 CCM 的 $+/-/.$ 符号表一阶一阶看，**从第几阶起一片全 `.`，前一阶就是 $q$**（同单序列看 ACF 在哪截断定 MA 阶）。本 demo `lag 1` 四格全显著、`lag≥2` 全 `.` → $q=1$。
> 对比 VAR：VAR 的 CCM 是缓慢**拖尾**、数不出截断点，只能靠 AIC/BIC。这正是两者定阶法不同的根源。

### 估计 第3步 条件 MLE 反推冲击（不能 OLS）
冲击看不见 → 不能像 VAR 那样 OLS，只能 MLE：

- **条件 MLE**：令 $a_0=0$，按 $\hat a_t=r_t-\theta_0+\Theta_1\hat a_{t-1}$ **递推**出每期冲击，再让高斯似然最大。本 demo 把 $\Sigma$ **解析集中并出**（$\hat\Sigma=\frac1T\sum\hat a_t\hat a_t'$），目标只剩 $\tfrac{T}{2}\ln|\hat\Sigma(\theta_0,\Theta_1)|$，用 `scipy.minimize` 优化 6 个参数。
- **精确 MLE**：把 $a_0$ 当未知量一起估，更准但更重（本 demo 用条件法，样本大时两者几乎一致）。
- **精修**：把 $|t|<1.96$ 的参数剔掉再估。本 demo 把 `MKT<-aSTK` 这一项剔为 0，**还原了"冲击单向传导"的结构**。
- $\Sigma$ 估出非对角 $\approx0.47$、同期相关 $\approx0.49$ → 同期联动还原。

### 检验 第4步 残差白噪声（与 VAR 完全相同）
- **多元 Ljung-Box $Q_k(m)$**：整体查残差 CCM 是否全 0，$df=k^2m-k^2q$。
- **残差 CCM 符号表**：按 $2/\sqrt{T}$ 判显著，应**几乎全 `.`**。
- 不合格 → 回第 2 步调 $q$。
> 注意：$T$ 大时，符号表里**个别格偶然显著**很正常（5% 水平下期望约 $5\%$ 的格数）——以**正式的 $Q_k(m)$ 检验为准**，别被一两个孤立的 `+/-` 吓回去。本 demo 16 格里 2 格显著（偶然期望 ≈0.8），$Q_k$ 通过 → 判充分。

### 预测 第5步 有限记忆（VMA 独有特点）
- $\hat r_T(1)=\theta_0-\Theta_1 a_T$（1 步还用得上最后一期冲击）；
- $\hat r_T(\ell)=\theta_0$，当 $\ell>q$（冲击用完，**立刻回均值**）；
- 误差协方差：$\hat\Sigma(1)=\Sigma$；$\hat\Sigma(\ell)=\Sigma+\Theta_1\Sigma\Theta_1'$（$\ell\ge2$，**之后恒定不再增长**）。
> 这是 VMA 区别于 VAR 的最直观证据：预测点在 $q$ 步后**拍平到均值**、误差标准差**跳一级即封顶**（VAR 则会一直缓慢增长）。见 `vma_fig4`（预测拍平）与 `vma_fig5` 右图（误差 SE 阶梯封顶）。

## 5. 判据一览

| 步骤 | 判据 | 通过条件 |
|---|---|---|
| 平稳 | 各分量 ADF（有限阶 VMA 恒平稳） | < 0.05（一般自动满足） |
| 可逆 | $\Theta_i$ 特征值模长 | 全 < 1 |
| 定阶 $q$ | 样本 CCM 符号表 | 从某阶起全 `.`，前一阶 = $q$ |
| 估计 | 条件/精确 MLE 收敛 | 似然最大、可逆 |
| 残差白噪声 | 多元 Ljung-Box $Q_k(m)$ | > 0.05（正式检验，优先级最高） |
| 残差无残留 | CCM 符号表 | 几乎全 `.`（个别格属偶然） |
| 预测·大小 | 各分量 RMSE vs 朴素 | 更小 → 更优 |
| 预测·分布 | 区间覆盖率 | ≈ 名义水平(95%) |

## 6. 易混点澄清

1. **截尾 vs 拖尾**：VMA 的 CCM **截尾**（$q$ 后骤为 0）→ 能数出 $q$；VAR 的 CCM **拖尾**（缓慢衰减不归零）→ 数不出、靠 AIC/BIC。**这是区分该上 VMA 还是 VAR 的第一眼判据。**
2. **为什么 VMA 不能 OLS**：OLS 要求回归元可观测；VMA 右边是看不见的冲击 $a_{t-i}$，必须先用"可逆 + 递推"把它们反推出来 → 天然是 MLE 问题。
3. **可逆性 ≠ 平稳性**：VMA 平稳几乎白送（有限阶必平稳）；真正的约束是**可逆**（冲击反推得回），对应 $\Theta_i$ 特征值在单位圆内。这与 VAR/AR 的"平稳=特征值在圆内"刚好是对偶的关注点。
4. **可识别性**：VMA 参数可能多解（例如整体符号/尺度的不定），需加约束才能唯一确定——这是 MA 类模型特有的坑。
5. **有限记忆**：预测超过 $q$ 步立刻回均值、误差方差封顶；别期待 VMA 像 VAR 那样做长程预测，它的"记忆"就只有 $q$ 期。
6. **条件 vs 精确 MLE**：条件法令 $a_0=0$、实现简单，样本大时和精确法几乎一样；样本短或根接近单位圆时，精确 MLE 更稳。

## 8. 全流程记忆口诀

> **平稳（自动）且可逆（$\Theta$ 特征值在圆内、冲击反推得回）的多元序列 → 数样本 CCM 截尾定 $q$（VAR 拖尾数不出、VMA 截尾数得出）→ 条件 MLE 反推冲击估 $\theta_0,\Theta_i$、精修剔不显著、注意可识别 → 残差查 $Q_k(m)$ 与 CCM（个别偶然格无妨，以 $Q_k$ 为准）→ 有限记忆预测（$>q$ 步即回均值、误差方差封顶）。**
> 骨架与 VAR 同；只有"CCM 截尾定阶"和"MLE 反推冲击"两处特色，根源都在"冲击看不见"。
