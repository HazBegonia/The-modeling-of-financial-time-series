## 0. 一句话理解 VAR

标量 **AR** 让一个序列看自己的过去；**VAR（向量自回归）就是把 AR 的"系数"换成"矩阵"，让 $k$ 个序列同时看彼此的过去。**

$$
\underbrace{r_t}_{k\times1}=\underbrace{\phi_0}_{k\times1}+\underbrace{\Phi_1}_{k\times k} r_{t-1}+\dots+\Phi_p r_{t-p}+\underbrace{a_t}_{k\times1},\qquad \mathrm{Cov}(a_t)=\Sigma
$$

- **$\Phi_i$ 的对角线** = 每个序列的"自身记忆"（像各自的 AR）；
- **$\Phi_i$ 的非对角线** = "谁带动谁"的**跨序列引导**（领先—滞后关系）；
- **$\Sigma$ 的非对角线** = "同一天一起动"的**同期相关**（无法用滞后解释，藏在噪声协方差里）。

本 demo 用 $k=2$：个股 `STK` + 大盘 `MKT`，复刻例 8.4 的两个招牌结论——**大盘单向引导个股**（$\Phi$ 非对角）、**同日齐涨齐跌**（$\Sigma$ 非对角）。

---

## 1. 核心概念速查

| 概念 | 含义 |
|---|---|
| 维数 $k$ | 同时建模的序列个数（本 demo $k=2$） |
| 阶 $p$ | 往回看几期（每期一个 $k\times k$ 矩阵 $\Phi_i$） |
| 简化式 reduced form | 右边只含**滞后**项，每个方程**回归元完全相同** → 可逐方程 OLS |
| 同期相关 | $\Sigma$ 的非对角元；同一天的联动，不靠滞后解释 |
| 单向引导 one-way lead | $A$ 的滞后进了 $B$ 的方程、但 $B$ 的滞后没进 $A$ 的方程（Granger 因果单向） |
| 伴随矩阵 companion | 把 VAR($p$) 摊平成 VAR(1) 的大矩阵；其特征值定平稳 |
| CCM | 交叉-相关矩阵（cross-correlation matrix），残差 CCM 全 0 才算白噪声 |

**关键认知**：VAR 把"多个序列的相互牵动"拆成两块——**能用过去解释的**塞进 $\Phi$（引导），**只发生在同一期的**塞进 $\Sigma$（同期相关）。单独对每个序列做 AR 会把这两块信息全丢掉。

## 2. 数据要求（喂进去之前先验收）

| 要求 | 怎么查 | 不满足怎么办 |
|---|---|---|
| **每个分量弱平稳** | 各分量 ADF / KPSS | 先差分；若存在协整，改用 VECM |
| **联合平稳（向量版）** | **伴随矩阵特征值全在单位圆内** | 不满足 → 差分 / 协整分析 |
| **等间隔、同步对齐** | 看时间索引 | 重采样 / 对齐 |
| **序列数别太多** | 数 $k$；参数 $\propto k^2 p$ | $k$ 大 → 参数爆炸 → 用更小 $p$ / 稀疏 VAR / 因子模型 |
| **噪声分布兜得住** | 残差 QQ / 峰度 | 厚尾换多元 $t$ |

> 一句话：**喂进 VAR 的应是——等间隔、各分量平稳且联合平稳（伴随特征值在单位圆内）、维数 $k$ 不至于让 $k^2p$ 个参数爆炸的多元序列。**

> **参数爆炸**就藏在信息准则的罚项 $k^2 i$ 里：$k$ 翻倍、罚项翻四倍——这是 VAR 实务里最先撞上的墙。

## 3. 五步建模流程（对照 demo）

```
VAR(2)造数据 ─→ 前提:各分量ADF + 伴随特征值<1 ─→ 定阶 p
                                                  │ M(i)逐阶似然比(式8.18) + AIC/BIC/HQ取最小
                                                  ↓
        向量递推预测 ←── 残差检验 ←──────────── 估计
     (点+区间,Psi权重)   Q_k(m)白噪声+CCM全'.'   逐方程OLS + 剔不显著项 + Granger单向引导
```

### 前提 第1步 平稳（书 8.2.2）
建模默认 $r_t$ 弱平稳。各分量先 ADF；**真正的向量版条件是把 VAR($p$) 摊成伴随矩阵，看它的特征值模长是否全 $<1$（全在单位圆内）**——直觉是"昨天的冲击要会衰减、不能越滚越大"。不平稳就差分或考虑协整。这是入场券。

### 定阶 第2步 决定 $p$（书 8.2.4）

**(2a) M(i) 逐阶似然比检验（式 8.18）**：一阶一阶地问"再加第 $i$ 阶矩阵 $\Phi_i$ 有没有用？"
$$M(i)=-\Big(T-k-i-\tfrac32\Big)\ln\frac{|\hat\Sigma_i|}{|\hat\Sigma_{i-1}|}\ \xrightarrow{H_0}\ \chi^2_{k^2}$$
- $H_0:\Phi_i=0$（加了没用）。$M(i)$ 大、超临界 → 这阶有用，保留；否则砍。
- 关键实现细节：**各阶 $\hat\Sigma_i$ 必须在同一公共样本上估**（本 demo 起点统一取 `START=PMAX`），否则行列式之比不可比。
- 读法：找"**最后一个显著阶**"。本 demo $i=1,2$ 极显著、$i\ge3$ 全不显著 → $p=2$。

**(2b) 信息准则 AIC / BIC / HQ（取最小）**
$$\mathrm{AIC}(i)=\ln|\hat\Sigma_i|+\frac{2k^2 i}{T},\quad \mathrm{BIC}(i)=\ln|\hat\Sigma_i|+\frac{k^2 i\ln T}{T},\quad \mathrm{HQ}(i)=\ln|\hat\Sigma_i|+\frac{2k^2 i\ln\ln T}{T}$$
- 第一项 $\ln|\hat\Sigma_i|$ = 拟合好不好（残差越小越好）；第二项 = 罚参数（含 $k^2i$，阶/序列越多罚越狠）。
- 两者相加取最小 → 自动在"拟合"与"省参数"间权衡。**BIC/HQ 罚更重，倾向更小阶**。本 demo 三准则都在 $i=2$ 见底。

> $M(i)$ 是"逐阶做假设检验"，AIC/BIC/HQ 是"打分选最低"，目的一样——定 $p$。

### 估计 第3步 算系数（书 8.2.4）
- **逐方程 OLS**：简化式 VAR 每个方程右边解释变量都一样 → 可以一个方程一个方程分开 OLS，简单且与 ML 渐近等价。
- 残差协方差 $\hat\Sigma=\frac1T\sum_t \hat a_t\hat a_t'$（式 8.19）。本 demo 估出非对角 $\approx0.61$、同期相关 $\approx0.60$ → **同步关系还原**。
- **精修（step b）**：把统计上不显著的滞后项（$|t|<1.96$）剔掉再重估，让模型更省参数。本 demo 精简后 **STK 方程仍留 MKT 滞后、MKT 方程仍无 STK 滞后**——单向引导结构被保住。
- **Granger 因果（8.1.2）**：单方程 F 检验"加入对方滞后有没有用"。本 demo `MKT→STK` 显著（$p\approx2e\text{-}15$）、`STK→MKT` 不显著（$p\approx0.5$）→ **单向引导被估成具体数字**。

### 检验 第4步 残差榨干没有（书 8.2.4）
- **多元 Ljung-Box $Q_k(m)$**：整体检验残差的 CCM 到 $m$ 阶是否全 0。
$$Q_k(m)=T^2\sum_{l=1}^m\frac{1}{T-l}\,\mathrm{tr}\!\big(\hat\Gamma_l'\hat\Gamma_0^{-1}\hat\Gamma_l\hat\Gamma_0^{-1}\big)\ \sim\ \chi^2,\ \ df=k^2m-k^2p$$
- **残差 CCM 符号表（+/−/.）**：每个滞后的互相关阵，按 $2/\sqrt{T}$ 两倍标准误判显著。**几乎全是 `.`** → 自身相关和互相带动都被吸收干净 → 模型充分。
- 不合格 → 回第 2 步升阶 / 改设定，迭代。

### 预测 第5步 合格后才能用（书 8.2.4 末）
- **1 步**：$\hat r_t(1)=\hat\phi_0+\hat\Phi_1 r_t+\dots+\hat\Phi_p r_{t-p+1}$；预测误差就是下期冲击 $a_{t+1}$，协方差 $=\Sigma$。
- **多步**：把已预测的值当已知，继续往后递推（plug-in）。
- **预测误差协方差**：用 MA($\infty$) 表示的 $\Psi$ 权重逐步累加——$\hat\Sigma(\ell)=\sum_{j=0}^{\ell-1}\Psi_j\Sigma\Psi_j'$，$\Psi_0=I$，$\Psi_\ell=\sum_{j=1}^{\min(\ell,p)}\Phi_j\Psi_{\ell-j}$。区间 $=\pm1.96\sqrt{\mathrm{diag}}$。
- 评估：各分量 RMSE vs 朴素 last-value、95% 区间覆盖率。

## 4. 判据一览（每步用什么 p 值 / 准则决策）

| 步骤 | 判据 | 通过条件 |
|---|---|---|
| 各分量平稳 | ADF p | < 0.05 |
| 联合平稳 | 伴随矩阵特征值模长 | 全 < 1（单位圆内） |
| 定阶（检验） | $M(i)$ p 值（$\chi^2_{k^2}$） | 取"最后一个显著阶" |
| 定阶（打分） | AIC / BIC / HQ | 准则最小（BIC/HQ 偏小阶） |
| 引导方向 | Granger F | < 0.05 → 存在引导 |
| 残差白噪声 | 多元 Ljung-Box $Q_k(m)$ | > 0.05 → 充分 |
| 残差无残留 | CCM 符号表 | 几乎全 `.` |
| 预测·大小 | 各分量 RMSE vs 朴素 | 更小 → 更优 |
| 预测·分布 | 区间覆盖率 | ≈ 名义水平(95%) |

## 5. 易混点澄清

1. **VAR vs 多个独立 AR**：分开做 AR 丢掉**跨序列引导**（$\Phi$ 非对角）和**同期相关**（$\Sigma$ 非对角）；VAR 把这两块一并建进来。
2. **引导（lead）≠ 同期相关**：引导能用**过去**解释（进 $\Phi$，可做 Granger 检验）；同期相关只发生在**同一期**（进 $\Sigma$，靠残差协方差看）。本 demo 两者都有，且是两码事。
3. **Granger 因果 ≠ 真因果**：Granger 只说"对方的过去能否改善本方程预测"，是**预测意义**上的引导，不等于结构/物理因果。
4. **平稳条件是"伴随特征值 < 1"，不是看 $\Phi_1$**：VAR($p$) 要先摊成伴随矩阵 VAR(1) 再看特征值；只盯单个 $\Phi_i$ 会判错。
5. **参数爆炸**：参数数 $\propto k^2p$。$k$ 一大就先撞墙——这正是 IC 罚项里 $k^2i$ 的现实含义；应对是压小 $p$、稀疏化、或转因子模型。
6. **M(i) 各阶必须同一样本**：比较相邻阶的 $|\hat\Sigma|$ 之比时，若样本长度不同则不可比；务必固定公共起点。

## 7. 全流程记忆口诀

> **联合平稳的多元序列（伴随特征值在单位圆内）→ 定阶 $p$（$M(i)$ 逐阶似然比 + AIC/BIC/HQ 取最小，注意 $k^2p$ 参数爆炸）→ 逐方程 OLS 估计、剔不显著项、看 $\Sigma$ 的同期相关与 Granger 的单向引导 → 残差查多元 Ljung-Box $Q_k(m)$ 与 CCM 符号表（全 `.` 才充分）→ 向量递推做多步预测（点 + $\Psi$ 权重区间），按大小/分布评估。**
> 每一步的"门槛"都是一个假设检验的 p 值或信息准则，不达标就回上一步。
