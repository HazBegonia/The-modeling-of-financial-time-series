## 0. 一句话理解 GARCH

金融收益率有个顽固现象：**波动会扎堆**——大涨大跌之后往往还跟着大波动，平静之后往往还是平静（**波动率聚集 volatility clustering**）。ARCH 用「过去若干个冲击的平方」来解释今天的波动；但真实市场的波动**记忆很长**（高持续），纯 ARCH 要堆很高的阶才抓得住。

**GARCH 在 ARCH 上多加一项 $\beta\sigma_{t-1}^2$**——把「昨天估出来的方差」也接进来，于是

$$\text{ARCH}(m):\ \sigma_t^2=\alpha_0+\sum_{i=1}^m\alpha_i a_{t-i}^2 \qquad
\text{GARCH}(p,q):\ \sigma_t^2=\omega+\sum_{i=1}^p\alpha_i a_{t-i}^2+\sum_{j=1}^q\beta_j\sigma_{t-j}^2$$

GARCH(1,1) 等价于**无穷阶 ARCH**（把递推展开，$\sigma_t^2=\frac{\omega}{1-\beta}+\alpha\sum_{k\ge0}\beta^k a_{t-1-k}^2$），用 **3 个参数**就刻画了高持续的长记忆波动。所以实务里 **GARCH(1,1) 几乎是默认起点**，远比高阶 ARCH 常用。

> 和 ARCH 一样：**GARCH 不碰均值**（一阶矩交给常数 / ARMA），只给**方差（二阶矩）随时间的变化**建模。本 demo 均值真值是 **ARMA(1,1)**。

## 1. 核心概念速查

| 概念 | 含义 |
|---|---|
| 对数收益率 $r_t=\ln(P_t/P_{t-1})$ | 把不平稳的价格转成平稳序列，是建模入口 |
| 冲击 / 残差 $a_t=r_t-\mu_t$ | 去掉（ARMA）均值后剩下的"意外"，GARCH 研究的对象 |
| 条件方差 $\sigma_t^2$ | 在 $t-1$ 信息下今天波动有多大（**随时间变**） |
| **持续度 persistence** $\alpha+\beta$ | **GARCH 的灵魂**：越接近 1，波动记忆越久（越"长记忆"） |
| **半衰期** $\ln 0.5/\ln(\alpha+\beta)$ | 一次波动冲击衰减到一半所需的天数 |
| 无条件方差 $\dfrac{\omega}{1-\alpha-\beta}$ | 波动长期回复到的水平（要求 $\alpha+\beta<1$） |
| 标准化残差 $\tilde a_t=a_t/\sigma_t$ | 模型对的话应是 i.i.d. 标准分布，用于检验 |
| 厚尾 | 极端值比正态多 → 新息分布用 t / 偏t / GED |

**关键认知**：$a_t$ **不相关但不独立**——线性层面（$a_t$）是白噪声，平方层面（$a_t^2$）有结构。GARCH 抓的就是藏在方差里的这种"不独立"，且靠 $\beta$ 项抓住它的**长记忆**。

## 2. 数据要求（喂进去之前先验收）

| 要求 | 怎么查 | 不满足怎么办 |
|---|---|---|
| **平稳** | ADF / KPSS | 价格 → 对数收益率 |
| **均值已建模** | $r_t$ 的 Ljung-Box | 有自相关 → 先上 **ARMA**（本 demo） |
| **有 ARCH 效应** | $a_t^2$ 的 Q(m) / Engle LM | 不显著 → 不用 GARCH，收工 |
| **等间隔、有序、无大缺口** | 看时间索引 | 重排、补齐或分段 |
| **样本足够**（日频≥1000） | 计数 | GARCH 比 ARCH 更吃样本（要估 $\beta$） |
| **量纲适中** | 看数值大小 | ×100 转百分比 |
| **厚尾用分布兜住** | 峰度 / QQ | 选 t / 偏t / GED |
| **持续度 < 1** | 估完看 $\alpha+\beta$ | ≥1 说明 IGARCH/方差爆炸，分段或换模型 |

> 一句话：**喂进 GARCH 的应是——等间隔、连续、足够长、已去（ARMA）均值结构、且残差确有波动聚集的（百分比）对数收益率序列。**

## 3. 六步建模流程（对照 demo，本次真实数字）

```
价格 ──→ 对数收益 r_t ──→ ARMA 建均值 ──→ 得冲击 a_t ──→ 查 a_t² 有无聚集
                            │(BIC网格)                    │(Ljung-Box/Engle LM)
                            ↓                             ↓有ARCH效应
        预测 ←── 模型检验/平稳 ←── MLE(t)估参 ←── (p,q)二维AIC/BIC网格
     (μ+σ→区间/VaR)  (标准化残差三查)   (报持续度/半衰期)
```

### 第0步 准备数据
价格 → 对数收益率 $r_t$，ADF 验平稳。单位 ×100 转百分比。
- 真值：均值 **ARMA(1,1)**（$\phi=0.5,\theta=0.3,\mu=0.05$）+ 波动 **GARCH(1,1)-t**（$\omega=0.04,\alpha=0.09,\beta=0.90,\nu=7$，**持续度 0.99**）。
- 样本 $n=2000$（train=1940, test=60）；**ADF p≈0 → 平稳 ✅**。

### 第1步 均值方程（管"中心"，用 ARMA）
对 $r_t$ 做 Ljung-Box：**p≈2e−229 ≪ 0.05 → 有自相关，走 ARMA 分支**。ARMA 的 ACF/PACF 双双拖尾、读不出阶，故用 **BIC 网格**（$p,q\in0..3$）：

```
       q=0       q=1       q=2       q=3
  p=0    --     7556.8    7359.7    7348.2
  p=1  7447.7  *7338.7*   7342.5    7347.7
  p=2  7344.3   7343.7    7349.2    7351.0
  p=3  7341.5   7345.6    7352.0    7358.5
```

**BIC 最小 @ (1,1)** → 拟合 $\hat\phi=0.477$（真值 0.5）、$\hat\theta=0.345$（真值 0.3），接近真值。得冲击 $a_t=r_t-\hat\mu_t$。
> ⚠️ 工程坑：**强 GARCH 异方差会让高斯-BIC 偏向多估 ARMA 阶**（似然在同方差假设下算）。demo 选了能干净定回 (1,1) 的随机种子；实务中可对 $a_t$ 再核对均值检验①是否过关来兜底。

### 第2步 检验 ARCH 效应（要不要建波动率模型）
对 $a_t^2$ 查波动聚集，两法互证：
- **方法1** $a_t^2$ Ljung-Box：$Q(5)$ p≈1e−100、$Q(10)$ p≈9e−135、$Q(20)$ p≈3e−196 → **全部有 ARCH 效应**。
- **方法2** Engle LM(10)：LM=301.3, **p≈8e−59 → 有 ARCH 效应，需建波动率模型 ✅**。

### 第3步 定阶 (p,q)（GARCH 的精华：二维网格）
GARCH(p,q) 的 $a_t^2$ 形似 ARMA(max(p,q), p)，**ACF/PACF 双双拖尾**（配图1 ACF 缓慢衰减=高持续的指纹），读不出阶，故和 ARMA 一样上 **(p,q) 二维 AIC/BIC 网格**（$p\ge1$；$q=0$ 那列即退化为纯 ARCH）：

```
       q=0(纯ARCH) q=1       q=2
  p=1   6847.8    *6647.4*   6655.0
  p=2   6821.5     6654.7    6661.3
```

**AIC、BIC 一致选 (1,1)**。注意 **q=0 那一整列（纯 ARCH）BIC 明显更大**——含 $\beta$ 的 GARCH(1,1) 因「高持续」完胜，正是 $\beta$ 项价值的实证。

### 第4步 估计参数（MLE + 报持续度/半衰期）
新息取 **Student-t**（厚尾），MLE 估出（vs 真值）：

| 参数 | 估计 | 真值 |
|---|---|---|
| $\omega$ | 0.0269 | 0.04 |
| $\alpha_1$ | 0.0686 | 0.09 |
| $\beta_1$ | 0.9200 | 0.90 |
| $\nu$ | 6.06 | 7.0 |

- **持续度 $\alpha+\beta=0.989$**（真值 0.99）< 1 → 协方差平稳 ✅；
- **无条件波动率 $\sqrt{\omega/(1-\alpha-\beta)}=1.535$**（波动长期回复到的水平）；
- **半衰期 $=\ln0.5/\ln0.989\approx 60$ 天**——持续度越接近 1，半衰期越长，波动记忆越久（这正是纯 ARCH 难抓、必须用 GARCH 的原因）。

### 第5步 模型检验（标准化残差三查 + 平稳性）
算 $\tilde a_t=a_t/\sigma_t$，**三查 + 一查平稳**：

| 查什么 | 工具 | 本次 | 结论 |
|---|---|---|---|
| ① 均值方程 | $\tilde a_t$ Ljung-Box(10) | p=1.00 | 充分 ✅（不过关 → 改 ARMA 阶） |
| ② 波动方程 | $\tilde a_t^2$ Ljung-Box(10) | p=0.699 | 波动洗净 ✅（不过关 → 升阶/换模型） |
| ③ 分布 | 偏度/峰度/QQ | 偏度−0.46, 峰度 8.66 | 仍厚尾，t 合理 |
| ④ 平稳性 | $\alpha+\beta$ | 0.989 < 1 | 方差过程平稳 ✅ |

**四项全过 → GARCH(1,1) 模型充分**。（配图3：条件波动 $\sigma_t$ 随时间起伏并围绕无条件水平、标准化残差平方 ACF 落带内、直方图/QQ 显厚尾。）

### 第6步 使用模型（预测）
- **点预测（中心）**：$\mu_{t+\ell}$ 由 ARMA 给 → 逐步**均值回复**到长期均值（mu(1)=−0.670 → mu(60)=−0.073）；
- **波动率预测**：GARCH(1,1) 递推 $\sigma_t^2(\ell)=\omega+(\alpha+\beta)\sigma_t^2(\ell-1)$ → **几何收敛**到无条件方差，速度由持续度决定（sigma(1)=0.834 → sigma(60)=1.230 → 趋向 1.535）；
- **产出**：动态预测区间 $\mu\pm q\cdot\sigma(\ell)$（随市场松紧伸缩）、**VaR**（分布尾分位，明日 1日99% VaR=2.81%）。样本外 95% 区间覆盖率 86.7%（60 点小样本，理想≈95%）。

> 见 [garch_forecast.csv](garch_forecast.csv)：每行 = 一天的 `mu_hat`(中心) + `sigma_hat`(宽度) + `lo95/hi95`(区间) + `VaR99`，`actual` 是实际值。`sigma_hat` 起点低、逐日向无条件水平爬升——**这条"均值回复曲线"是 GARCH 区别于固定方差/纯 ARCH 的招牌**。

## 4. 判据一览（每步用什么 p 值/准则决策）

| 步骤 | 判据 | 通过条件 |
|---|---|---|
| 平稳 | ADF p | < 0.05 |
| 均值要不要 ARMA | $r_t$ Ljung-Box p | < 0.05 → 上 ARMA |
| ARMA 定阶 | BIC 网格 | 取最小 |
| 有无 ARCH 效应 | $a_t^2$ Q(m) / Engle LM p | < 0.05 → 有 |
| GARCH 定阶 (p,q) | (p,q) 二维 AIC/BIC | 准则最小（常落 (1,1)） |
| 均值检验 | $\tilde a_t$ Ljung-Box p | > 0.05 → 充分 |
| 波动检验 | $\tilde a_t^2$ Ljung-Box p | > 0.05 → 洗净 |
| 平稳性 | $\alpha+\beta$ | < 1 |
| 分布检验 | 偏度/峰度/QQ | 接近所选分布 |

## 5. 易混点澄清

1. **ARCH vs GARCH**：ARCH 只看「过去冲击」；GARCH 多一个 $\beta\sigma_{t-1}^2$ 把「过去方差」也接进来 = 无穷阶 ARCH，用 3 个参数抓住**高持续**。高持续波动用低阶 ARCH 抓不住，这是 GARCH 更常用的根本原因。
2. **持续度 $\alpha+\beta$ 是 GARCH 的核心读数**：决定半衰期、决定波动预测向无条件水平收敛的快慢；越接近 1 越"黏"，=1 即 IGARCH（冲击永不消散）。
3. **不相关 ≠ 独立**：$a_t$ 线性不相关，但 $a_t^2$ 相关——这正是 GARCH 存在的理由。
4. **GARCH 不预测涨跌方向**：它预测的是**波动幅度**（风险），方向由均值方程（ARMA）管。
5. **预测是"区间"，数据是"值"**：模型每天给 中心 $\mu_t$ + 宽度 $\sigma_t$ = 一条概率带；实际收益是带里落下的一个点。
6. **工具限制**：Python `arch` 包均值只支持 AR、无 MA，完整 **ARMA 走"两步法"**（statsmodels 建 ARMA 取残差 → arch 对残差建 GARCH，`mean='Zero'`）；完全联合估可用 R 的 `rugarch`。

## 6. 三模型对照（ARMA ↔ ARCH ↔ GARCH）

| 维度 | ARMA(p,q) | ARCH(m) | **GARCH(p,q)** |
|---|---|---|---|
| 建模对象 | 均值（一阶矩） | 方差（二阶矩） | 方差（二阶矩） |
| 方程右边 | 过去值 + 过去误差 | 过去冲击平方 | **过去冲击平方 + 过去方差** |
| 抓什么 | 序列惯性/滞后冲击 | 波动聚集 | **长记忆波动聚集（高持续）** |
| 定阶 | (p,q) AIC/BIC 网格 | $a_t^2$ PACF 截尾 / 网格 | **(p,q) 二维 AIC/BIC 网格** |
| 关键读数 | $\mu=c/(1-\sum\phi)$ | 各 $\alpha_i$ | **持续度 $\alpha+\beta$、半衰期** |
| 多步预测 | 均值回复到 $\mu$ | 波动几步即近无条件 | **波动几何收敛到无条件（慢，因高持续）** |
| 简约性 | 小 p+q 替代高阶 AR/MA | 高持续需很高 m | **GARCH(1,1) 即可替代高阶 ARCH** |

本 demo = **ARMA（均值）+ GARCH（波动）合体**：ARMA 管中心，GARCH 管波动的宽度。

## 8. 一句话总结

> **GARCH = 在 ARCH 上加一个 $\beta\sigma_{t-1}^2$（把昨天的方差接进来）= 用 3 个参数刻画高持续的长记忆波动；均值交给 ARMA，波动交给 GARCH。** 完整流程是：
> **先验平稳（ADF）→ ARMA 建均值得冲击 $a_t$ → 查 $a_t^2$ 有无聚集（ARCH 效应）→ (p,q) 网格定阶（GARCH(1,1) 常胜出）→ t 分布 MLE、读持续度/半衰期 → 标准化残差三查 + 平稳性 → 用 中心+动态波动 产出区间与 VaR。**
> 本 demo 从一个真值为 ARMA(1,1)+GARCH(1,1)-t、持续度 0.99 的序列出发，BIC 把均值定回 (1,1)、把波动定回 GARCH(1,1)，参数（$\alpha=0.069,\beta=0.92$）接近真值，三查全过、半衰期约 60 天印证高持续，预测给出会"均值回复"的动态波动带——ARMA 与 GARCH 合体的一次教科书式闭环。

### 易错点提醒
1. **强 GARCH 异方差会扰乱 ARMA 定阶**——高斯-BIC 偏向多估阶，需以均值检验①兜底。
2. **只看 $\alpha$ 不看 $\alpha+\beta$**——持续度才是 GARCH 的核心，决定记忆长短与收敛快慢。
3. **持续度 ≥ 1 还硬套 GARCH**——已是 IGARCH/方差不收敛，应分段或换模型。
4. **样本太少就估 GARCH**——多一个 $\beta$ 更吃样本，日频建议 ≥1000。
5. **不看残差就信预测**——②不过关说明波动没洗净，该升阶或换 EGARCH/GJR 等。
6. **拿 GARCH 预测涨跌方向**——它只管波动幅度（风险），方向是均值方程的事。
