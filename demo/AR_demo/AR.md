> 本文把 AR 模型的核心理论和 demo 的每一步对应起来，并附上本次真实运行的数字。

---

## 1. AR 模型是什么

**AR(p)（p 阶自回归模型）**：当前值由它自己过去 p 个值的线性组合加上一个白噪声扰动构成。

$$
y_t = c + \phi_1 y_{t-1} + \phi_2 y_{t-2} + \dots + \phi_p y_{t-p} + \varepsilon_t,\quad \varepsilon_t \sim \text{白噪声}(0,\sigma^2)
$$

- **核心思想**：用"过去预测未来"，"自回归"就是把变量对它自己的滞后项做回归。
- **长期均值**：$\mu = \dfrac{c}{1-\phi_1-\dots-\phi_p}$。序列围绕 $\mu$ 上下波动，这就是**均值回复（mean-reversion）**。
- 本 demo 真值：**AR(2)**，$y_t = 2.0 + 0.5\,y_{t-1} + 0.3\,y_{t-2} + \varepsilon_t$，$\sigma=1.0$，理论均值 $\mu = 2/(1-0.5-0.3)=10.0$。

### 弱平稳条件（建模前提）

AR 模型要求序列**弱平稳**（均值/方差不随时间变化、自协方差只与时间间隔有关）。AR(2) 的平稳条件是：

$$
\phi_1+\phi_2<1,\qquad \phi_2-\phi_1<1,\qquad |\phi_2|<1
$$

demo 取 $\phi_1=0.5,\phi_2=0.3$，三条都满足 → 平稳。不平稳的序列要先**差分**再建模（那就成了 ARIMA 的 I）。

## 3. 各步骤详解 + 本次运行结果

### 步骤 0 · 数据生成
用已知参数的 AR(2) 递推生成 500 个点，前 300 个为**烧入期（burn-in）丢弃**（消除初始值影响，让序列进入稳态）。留最后 `h=50` 个点做**样本外测试**，其余 450 个做训练。建模时"假装不知道阶数"，最后再揭晓真值验证。

### 步骤 1 · 平稳性检验（ADF）
**ADF（增广 Dickey–Fuller）检验**：原假设 $H_0$ = "有单位根（不平稳）"。p < 0.05 → 拒绝 → 平稳。

- 本次结果：**ADF = −6.432，p = 1.69e−08 ≪ 0.05 → 平稳，可建模 ✅**
- 配图 [ar_fig1_identify.png](ar_fig1_identify.png)：序列图 + ACF + PACF。

### 步骤 2 · 定阶（本 demo 的精华：三法交叉验证）

定阶 = 确定 p 该取几。demo 用三种方法互相印证，最后投票：

**2a · PACF 截尾法** —— AR 模型的识别口诀：

| | ACF（自相关） | PACF（偏自相关） |
|---|---|---|
| **AR(p)** | 拖尾（tails off，缓慢衰减） | **p 阶后截尾**（cuts off，突然掉进置信带） |
| MA(q) | q 阶后截尾 | 拖尾 |
| ARMA | 拖尾 | 拖尾 |

→ 看 PACF 在第几阶之后掉进 $\pm 1.96/\sqrt{n}$ 置信带，那个阶就是 p。本次：**最后显著 lag = 2**。

**2b · AIC / BIC 信息准则网格** —— 逐阶拟合，选准则**最小**的阶。
- AIC 偏向拟合优度，BIC 对复杂度惩罚更重（更"吝啬"，倾向选更小的阶）。
- ⚠️ **关键工程细节**：网格里用 `hold_back=MAXLAG`（[ar_demo.py:89](ar_demo.py#L89)），保证不同阶在**同一样本量**上拟合，IC 才有可比性。
- 本次：AIC 最小 @ p=2，BIC 最小 @ p=2。

**2c · 过拟合系数显著性法** —— 故意拟合一个偏大的 AR(6)，看哪些滞后项系数显著（p<0.05），不显著的高阶项应剔除。本次：显著的最高滞后 = 2。

**投票汇总**：`PACF=2 | AIC=2 | BIC=2 | 显著性=2` → **四法一致，最终定阶 p = 2（以 BIC 为准）**。
配图 [ar_fig2_order.png](ar_fig2_order.png)：AIC/BIC 随阶数变化，都在 p=2 处见底。

### 步骤 3 · 拟合
用定出的 p=2 拟合，估计参数 vs 真值：

| 参数 | 估计值 | 真值 |
|---|---|---|
| const | +2.183 | 2.0 |
| φ₁ (y.L1) | +0.569 | 0.5 |
| φ₂ (y.L2) | +0.207 | 0.3 |

估计值都很接近真值 → 模型抓对了结构。

### 步骤 4 · 残差诊断（判断模型是否充分）
**好模型的残差应近似白噪声**（无规律可挖、信息已榨干）。

- **Ljung–Box 检验**：$H_0$ = "残差无自相关"。p > 0.05 → 不能拒绝 → 残差是白噪声 ✅。
  本次：**Ljung-Box(10) p = 0.955 ≫ 0.05 → 残差无自相关，模型充分**。
- 残差均值 ≈ −0.000，标准差 ≈ 1.012（贴近真值 σ=1.0）。
- 配图 [ar_fig3_residuals.png](ar_fig3_residuals.png)：残差时序图（无趋势）、残差 ACF（全在带内）、直方图（近正态）、Q-Q 图（贴直线）。

### 步骤 5 · 样本外预测 + 评估
对未参与训练的最后 50 个点预测并评估：

- **RMSE = 1.833，MAE = 1.529**
- **朴素基准（last-value，用最后一个值预测）RMSE = 1.918** → **AR 更优 ✅**
- 配图 [ar_fig4_forecast.png](ar_fig4_forecast.png)：可清楚看到**均值回复**——多步预测逐渐收敛到长期均值 μ=10，置信区间随步数变宽（预测越远越不确定）。

## 4. 关键概念速查

- **ACF 拖尾 + PACF 截尾 → 认 AR；截尾的阶数 = p。** 这是肉眼定阶的第一直觉。
- **AIC vs BIC**：都越小越好；BIC 惩罚更重、更偏好简约模型。两者矛盾时，demo 以 BIC 为准。
- **`hold_back` 为什么重要**：比较不同阶的 IC 必须在同一样本量下，否则数字不可比。
- **残差白噪声 = 模型充分**的判据；若残差仍有自相关，说明阶数不够或结构没选对。
- **均值回复**：AR 的长期预测一定收敛到 μ，不会发散（平稳的体现）；因此 AR 适合**短期**预测，长期预测退化为"猜均值"。
- **朴素基准**：任何预测模型都该和 last-value / 均值这种 baseline 比，赢了才有意义。

## 6. 一句话总结

> **AR 模型 = 用自己的过去预测自己的未来。** 完整流程是：
> **先验平稳（ADF）→ 用 ACF/PACF + AIC/BIC + 系数显著性定阶 → 拟合参数 → 查残差是否白噪声 → 做样本外预测并跑赢 baseline。**
> 本 demo 从一个真值为 AR(2) 的序列出发，四法一致地把阶数定回了 2，参数估计接近真值，残差通过白噪声检验，预测优于朴素基准——一次教科书式的闭环验证。

### 易错点提醒
1. **不检验平稳性就直接建模**——不平稳要先差分。
2. **定阶只靠一种方法**——单一指标容易误判，多法交叉更稳。
3. **比较 IC 时样本量不一致**——记得 `hold_back`。
4. **不看残差就信预测**——残差有自相关说明模型没拟合充分。
5. **拿 AR 做超长期预测**——它只会收敛到均值，远期没有信息量。
